What is GenOrca?

GenOrca helps anyone create emotional, unforgettable content
using generative AI and real-time 3D tools like Unreal Engine.

From Claude-based agents to fully orchestrated asset placement,
GenOrca lets you focus on creativity, not complexity.

Feature Highlights

  • Multi-Client Integration (Claude, Cursor, VS Code, and more)
  • MCP Server: LLM Agents can control Unreal Editor via Python scripting
  • Unreal-MCPython: The first public release of the unreal-mcp project
